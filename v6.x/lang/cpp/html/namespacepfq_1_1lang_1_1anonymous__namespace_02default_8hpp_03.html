Evaluate to Pass Qbuff, or forward the packet to the given device.

It evaluates to Drop, depending on the value returned by the predicate. Example:

tap ("eth1", is_udp) >> kernel

Logically equivalent to:

unless (is_udp, forward ("eth1") >> drop ) >> kernel

Only a little bit more efficient.
