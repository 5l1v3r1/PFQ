auto pfq::lang::experimental::anonymous_namespace{experimental.hpp}::steer_gtp_usr
Initial value:
= [] (const char *net, int prefix)
{
return function("steer_gtp_usr", ipv4_t{net}, prefix);
}

Dispatch the packet across the sockets.

Dispatch with a randomized algorithm that maintains the integrity of per-user flows on top of GTP tunnels (Control-Plane packets are broadcasted to all sockets). Example:

steer_gtp_usr ("10.0.0.0", 8)
