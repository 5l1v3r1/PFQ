Detailed Description

PFQ: the socket.

This class is the main interface to the PFQ kernel module. Each instance handles a socket that can be used to receive from and transmit packets to the network.

bool pfq::socket::send_raw ( const_buffer  pkt,
int  ifindex,
int  qindex,
uint64_t  nsec,
unsigned int  copies,
bool  async = false,
int  queue = any_queue 
)
inline

Schedule a packet transmission.

The packet is copied into a Tx queue. If 'async' is true and 'queue' is set to any_queue, a TSS symmetric hash function is used to select the Tx queue. The packet is transmitted at the given timestamp by a PFQ kernel thread. Otherwise the queue is flushed every 'fhint' packets. A timestamp of 0 nanoseconds means immediate transmission.
