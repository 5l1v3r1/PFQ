PFQ

Network Monitoring Framework for Multi-core Architectures

Download this project as a .zip file Download this project as a tar.gz file

What is PFQ?

PFQ is a multi-language network monitoring framework designed for the Linux Kernel 3.x. It is highly optimized for multi-core processors, as well as for network devices equipped with multiple hardware queues (i.e. Intel 82599 10G).

PFQ consists in a Linux Kernel module and user-space libraries, for the C, C++11 and Haskell languages.

It works with vanilla drivers and provides a script (pfq-omatic) designed to obtain accelerated versions of the drivers away from their source codes.

Features

  • 10-Gbit Line-rate (14,8Mpps) with Intel ixgbe vanilla driver.
  • Socket groups allow for concurrent monitoring of multi-threaded applications.
  • Per-group packet steering through randomized hashing algorithms or deterministic packet classifications.
  • Per-group Berkeley and VLAN filters.
  • Extensible framework for in-kernel functional monitoring (continuation passing style).
  • User-space native bindings for the C, C++11 and Haskell languages.
  • PFQ-omatic, a script that allows to compile drivers and get accelerated versions.
  • Accelerated pcap library.

Performance

PFQ performance is highly dependent on the hardware in use.

Running on top of the 2.66Ghz 6-core Xeon equipped and the Intel 82599 10G controller, PFQ can steer to user-space about 14,8 Million packets per second deploying 3 kernel threads.

Details of performance comparison are available here (old version): http://netgroup.iet.unipi.it/software/pfq/performance.html

Publication

We received the Best-Paper-Award at PAM2012 in Vienna for the paper "PFQ: a Novel Engine for Multi-Gigabit Packet Capturing With Multi-Core Commodity Hardware"

Source code

The package provides the source code of the PFQ kernel module, user-space libraries for the C, C++11 and Haskell languages and a set of diagnostic tools.

Gcc/g++ 4.7 (or higher), the Haskell platform, Cmake and make are required for the compilation.
